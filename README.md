## 搬瓦工带宽体系一眼看懂：三条产品线的本质区别

搬瓦工目前把套餐分成三大系列：**Basic VPS**、**E-Commerce VPS**（就是大家熟悉的CN2 GIA-E）、**Ultra VPS**（香港/日本高端套餐）。

**Basic VPS**：端口速度1Gbps，走CN2 GT线路。CN2 GT是电信的次级优化线路，晚高峰会有一定拥塞，丢包率不至于崩溃但明显比GIA差一截。1TB月流量，年付$49.99，可以迁移到9个机房。纯粹入门用，或者学Linux、跑脚本、不在意延迟，选这个最省钱。

所以，**带宽不等于体验**。香港1Gbps可能比洛杉矶2.5Gbps快得多，因为路由更短。但大多数建站和日常用途，洛杉矶CN2 GIA-E已经够用，没必要多花5倍价格。

## 搬瓦工带宽的几个容易踩的坑

**坑一：Basic VPS迁到CN2机房流量变三分之一**。买之前没注意，迁移之后发现流量消耗速度突然快了，其实是这个机制。想用CN2线路又要保住流量，直接买E-Commerce。

**坑二：香港套餐月流量少但单价高**。香港入门款500GB流量，$89.99/月。流量超了会怎样？超出后带宽会降速，不会直接断，但体验会明显下滑。如果你的用途流量消耗大，香港性价比其实不高，它的价值在延迟而不在带宽总量。

**坑三：带宽标注是端口速度不是保证速度**。2.5Gbps是端口上限，不代表你一直能跑满。在服务器资源不拥挤的时候，跑满几乎没有问题；高峰期会有合理范围内的波动，但CN2 GIA的稳定性在这价位段算好的了。
